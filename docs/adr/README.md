# Architecture Decision Records (ADRs)

Cada ADR registra **uma decisão técnica importante**: o contexto, as opções consideradas, a decisão e
suas consequências. Decisões mudam com o tempo — quando isso ocorrer, crie um novo ADR que
**substitui** (supersedes) o anterior, em vez de apagar o histórico.

## Formato

```
# ADR-XXXX — Título
Status: Proposto | Aceito | Substituído por ADR-YYYY
Data: AAAA-MM-DD

## Contexto
## Decisão
## Consequências (positivas / negativas)
## Alternativas consideradas
```
